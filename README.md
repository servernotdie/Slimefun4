# Slimefun4 on Folia
Thử nghiệm chạy Slimefun4 trên Folia

Dự án này được sửa đổi dựa trên [nhánh main của Slimefun4 bản việt hóa](https://github.com/servernotdie/Slimefun4), chỉ thêm hỗ trợ chạy trên nền tảng Folia

**Khi gửi issue vui lòng xác nhận vấn đề chỉ tồn tại trong phiên bản sửa đổi này**

## Lấy bản build
